Sample applications


Here's a list of some PixelLight sample applications ordered from simple to complex:

05NetworkChat

Files
This OS-console based application demonstrates how a simple chat server/client can be written.

45ScriptApplication

Files

 Sample showing how to use "PLEngine::ScriptApplication" in order to load in and execute a scripted stand-alone application. The used script is a Lua port ("45ScriptApplication.lua") of the C++ sample 60Scene.

50RendererTriangle

Files

 This is a very basic sample showing you an amazing rotating triangle created by using a renderer vertex buffer.

51RendererTexturing

Files

 This is a very basic sample showing you an amazing rotating textured quad created by using a renderer vertex buffer.

52RendererMesh

Files

 This is a very basic sample demonstrating how to use meshes created with the mesh creator.

53RendererMultiView

Files

 This sample will use three frames. In two frames the triangle and teapot sample scenes will be shown. The main frame consists of three scenes where each is rendered using another viewport.

54RendererRTT

Files

 This is sample will show you how to render to texture (RTT). The teapot from the mesh sample is rendered to a texture which is used by some colorful quadrangles similar to the triangle sample. Also multi render targets (MRT) are shown. Using MRT you can render to multiple textures at the same time using a fragment shader.

55RendererTessellation

Files

 Basic tessellation example. DirectX 11 compatible GPU, required!

56RendererInstancing

Files

 Basic instancing example.

60Scene

Files

 Basic PixelLight sample showing how to setup a scene. The light position is changed using the RTTI attribute interface. Within the PixelLight SDK, there'a also a Lua port ("45ScriptApplication.lua") of this C++ sample.

61Sound

Files

 This sample demonstrates sound playback and time scale.

62Physics

Files

 This sample demonstrates how to use physics plugins within a PixelLight application. Because this sample is quite simple it is in fact physics API independent. The information text is drawn using an own special scene node.

63PhysicsRagdoll

Files

 Very similar to 62Physics, but instead of a small boring box, a cool ragdoll is thrown into the scene.

64Picking

Files

 Example showing how picking works.

65PostProcess

Files

 This sample will show you how to use post processing in your applications for effects like bloom, blur etc. Furthermore, you can see how the PixelLight file system is used to find some files - all available post processing effects are enumerated automatically.
Requires at least a DirectX 9 compatible GPU!

66IngameGUI

Files

 Simple sample showing how own ingame GUIs can be created by using PLGui

67Game2D

Files

 This visual and gameplay milestone will show you that it is also possible to create funny old school 2D games with PixelLight

70SPARK

Files

 The particle effects are defined within the SPARK_PL plugin which integrates the free open source particle engine "SPARK" into PixelLight.

71libRocket

Files

 Sample showing how to use the "PLlibRocket"-project which integrates the free open source HTML/CSS game interface middleware "libRocket" into PixelLight
