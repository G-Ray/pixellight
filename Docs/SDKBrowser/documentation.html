Documentation


The documentation will introduce you to the PixelLight API and will give you some hints to use it in the proper way. The framework codes are well documented, but we suggest to use this documentation during the development for further reading. PixelLight is also shipped together with sample programs. It is recommended to use these examples together with the documentation as a basis for own experiments. If you are new to 3D graphics or lack the technical background to some topics, e.g. you don't understand words like camera frustum, matrices, quaternions etc., note that this document won't explain these details, but only framework related things and short introductions, which will give you an idea of how it's done! A full explanation of all details would blow up the documentation and is beyond its scope. For more technical background you should use the internet or other references.
