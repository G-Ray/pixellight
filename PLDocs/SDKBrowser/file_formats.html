File formats

Here’s a list of all special PixelLight file formats ordered by the projects introducing them:

PLCore (see PLCore.pdf for more information)

Extension DTD Example Description
plugin plugin.dtd Example.plugin XML plugin format
cfg cfg.dtd Example.cfg XML configuration/settings format
loc loc.dtd Example.loc XML localization format
chunk - - Binary chunk format


PLMath (see PLMath.pdf for more information)

Extension DTD Example Description
graph graph.dtd Example.graph XML graph format
path path.dtd Example.path XML path format


PLGui (see PLGui.pdf for more information)

Extension DTD Example Description
gui gui.dtd Example.gui XML GUI format which is used to save and load GUI interfaces


PLRenderer (see PLRenderer.pdf for more information)

Extension DTD Example Description
plt plt.dtd Example.plt XML texture information format which holds additional settings for each texture like whether or not texture compression is allowed for a texture
tani tani.dtd Example.tani XML texture animation format where you define texture animations like rotating, sliding etc. textures
mat mat.dtd Example.mat XML material format holding material information
plfx plfx.dtd Example.plfx XML effect format holding effect information


PLMesh (see PLMesh.pdf for more information)

Extension DTD Example Description
mesh - - Binary chunk based mesh format which also stores animation data
xmesh - - XML version of the mesh-format


PLScene (see PLScene.pdf for more information)

Extension DTD Example Description
scene scene.dtd Example.scene XML scene format
sr sr.dtd Example.sr XML scene renderer format
skin skin.dtd Example.skin XML skin format of the SNMesh scene node to ’overwrite’ the default materials of the used mesh
sky sky.dtd Example.sky XML sky format of the SNSky scene node
terrain terrain.dtd Example.terrain XML terrain format of the SNTerrain scene node


PLCompositing (see PLCompositing.pdf for more information)

Extension DTD Example Description
pp pp.dtd Example.pp XML post process format


PLEngine

Extension DTD Example Description
cam cam.dtd Example.cam XML camcorder format


PLPhysics (see PLPhysics.pdf for more information)

Extension DTD Example Description
ragdoll ragdoll.dtd Example.ragdoll XML ragdoll format of the SNRagdoll scene node



(XML can be edited with a simple text editor)
