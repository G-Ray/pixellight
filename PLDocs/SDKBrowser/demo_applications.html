Demo applications


Here's a list of the PixelLight demo applications ordered from simple to complex:

 

PLDemoRendererTriangle

Files

 This is a very basic demo showing you an amazing rotating triangle created by using a renderer vertex buffer.

PLDemoRendererTexturing

Files

 This is a very basic demo showing you an amazing rotating textured quad created by using a renderer vertex buffer.

PLDemoMeshTeapot

Files

 This is a very basic demo demonstrating how to use meshes created with the mesh creator and
a simple light.

PLDemoRendererRTT

Files

 This is demo will show you how to render to texture. (RTT) The teapot from the teapot demo is rendered to a texture which is used by some colorful quadrangles similar to the triangle demo. Also multi render targets (MRT) are shown. Using MRT you can render to multiple textures at the same time using a fragment shader. (MultiRenderTargets.cg)

PLDemoRendererMultiView

Files

 This demo will use three frames. In two frames the triangle and teapot demo scenes will be shown. The main frame consists of three scenes where each is rendered using another viewport.

PLDemoSimpleScene

Files

 Basic PixelLight demo showing how to setup a scene. The light position is changed using the RTTI attribute interface.

PLDemoPicking

Files

 Demo showing how picking works.

PLDemoPhysicsBasic

Files

 This demo demonstrates how to use physics plugins within a PixelLight application. Because this demo is quite simple it is in fact physics API independent. The information text is drawn using an own special scene node.

PLDemoPhysicsRagdoll

Files

 Very similar to PLDemoPhysicsBasic, but instead of a small boring box, a cool ragdoll is thrown into the scene.

PLDemoNetworkChat

Files
This OS-console based application demonstrates how a simple chat server/client can be written using PLNetwork.

PLDemoSoundBasic

Files

 This demo demonstrates sound playback and time scale.

PLDemo2DGame

Files

 This visual and gameplay milestone will show you that it is also possible to create funny old school 2D games with PixelLight

PLDemoIngameGUI

Files

 Simple demo showing how own ingame GUIs can be created by using PLGui



For the following application you need at least a DirectX 9 compatible GPU!

PLDemoPostProcess

Files

 This demo will show you how to use post processing in your applications for effects like bloom, blur etc. Furthermore, you can see how the PixelLight file system is used to find some files - all available post processing effects are enumerated automatically.
