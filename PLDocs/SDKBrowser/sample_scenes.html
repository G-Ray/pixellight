Sample scenes


Here is a list of the PixelLight sample scenes in random order. The scenes can, for instance, be viewed by using the PLViewer within the 'Tools' directory.

 

AnchorCamera.scene

exe files

Shows how to attach a camera to a scene node.

AutomaticMeshes.scene

exe files

Shows how to create automatic meshes during runtime.

AutomaticTextures.scene

exe files

Shows how to create automatic textures during runtime.

CameraMoveController.scene

exe files

Shows how to attach a controller scene node modifier to a camera to be able to move it around by using the mouse.

CharacterAnimation.scene

exe files

Shows how to attach an animation scene node modifier to be able to control skeleton animations by using the scene system.

MoveController.scene

exe files

Shows how to use a controller scene node modifier to move around a scene node.

MultiCameras.scene

exe files

Scene with multiple cameras.

OrbiterCamera.scene

exe files

Orbiter camera example.

PLParticleGroups.scene

exe files

The particle effects are defined within the PLParticleGroups plugin.


Mipmaps.scene

exe files

Mipmaps visualisation.

ReflectionCubeMap.scene

exe files

Static reflection cube map.

SceneNodeModifiers.scene

exe files

Multiple scene node modifiers in action.

PhysicsCharacterController.scene

exe files

Physics character controller.

PhysicsFirstPersonController.scene

exe files

Physics first person controller.

PhysicsManiacMarble.scene

exe files

Another simple physics character controller sample scene.

PhysicsMouseInteraction.scene

exe files

This scene shows you how to implement mouse physics interaction.

  • Hold the third mouse button over an physics object to pick it, use the mouse wheel to pull/push the object towards/away
  • While picking, hold shift/strg-key to increase/decrease the picking force, hit the alt key to 'kick away' the picked object

PhysicsJoints.scene

exe files

Another simple physics character controller sample scene.

Scripts.scene

exe files

Simple scene with a running Lua script manipulating the scene.
